# 📅 校園王 (CampusKing) v6.4

> 專為高中與大學生設計的 Web OS 個人化校園生活管理助手。
> 校園生活，一手掌握。

## 🚀 專案簡介

**校園王 (CampusKing)** 是一個基於 Web 技術開發的校園生活管理系統 (Web App)。
採用創新的「Web OS」架構，將課表、成績、記帳、行事曆等功能模組化為獨立的 APP，提供使用者如原生作業系統般流暢的操作體驗。系統整合 Firebase 雲端資料庫，實現跨裝置資料同步與即時備份。

---

## 🚀 完整功能模組 (All Features)

### 📅 1. 智慧課表 (Schedule)

- **直觀互動編輯**：
  - **點擊即編**：點擊格子即可新增/編輯課程，支援「連選模式」(點頭尾自動選取)。
  - **連選模式**：支援「點頭選尾」操作，先點第 1 節、再點第 3 節，系統自動建立 1-3 節連堂課程。
- **雙視圖切換**：
  - **單日清單**：清楚顯示當天的課程、教室與老師。
  - **完整週課表**：傳統表格視圖，支援自動合併連堂課程 (Rowspan)。
- **智慧排課**：
  - 支援輸入 `1-3` 自動建立三堂連堂課程。
  - 自訂節次時間設定 (上課/下課分鐘數)。
  - 支援特殊節次代碼 (0-9, A-D, N)。
- **實用工具**：
  - **圖片匯出**：一鍵將週課表轉存為高畫質 PNG 圖片。
  - **上課提醒** 🔔：結合瀏覽器通知 API，於課程前 10 分鐘自動發送推播。
  - **課程標籤**：自動根據修別 (必修/選修) 標示不同顏色。

### 2. 🎨 沉浸式視覺體驗

- **動態主題**：每個功能 APP (課表、成績、記帳...) 皆擁有專屬的漸層背景色調。
- **深色模式**：完整支援 Dark Mode，背景自動切換為深色系漸層，舒適護眼。

### 💯 3. 成績管理 (Grade Manager)

- **深度雙身分核心**：
  - 支援「通識、核心、專業、自由選修」等學分分類統計。
  - **自主學習支援**：新增「自主學習」標記功能，勾選後該課程**僅計算學分 (歸類為自由選修)**，不納入學期加權平均 (GPA) 計算。
- **多維度成績紀錄**：
  - **平時考/小考**：獨立紀錄各科目的多次小考成績。
  - **段考/期中考**：紀錄重大考試成績。
  - **學期總成績**：計算最終 GPA 與學分。
- **智慧分析**：
  - **GPA 計算機**：自動計算學分加權平均。
  - **畢業進度條**：視覺化顯示距離畢業門檻的學分達成率。
  - **趨勢圖表**：繪製歷年成績折線圖，並自動標示 60/80 分參考線。

### 🎒 4. 作業成績 (Homework) [New]

- **作業清單**：
  - 條列式管理待繳作業與考試日期。
  - 支援一鍵標記「已完成 / 未完成」狀態，完成後自動刷淡並加上刪除線。
- **智慧輸入切換**：
  - **選單模式**：自動讀取週課表科目，快速選擇不用打字。
  - **手寫模式**：遇到非課表科目 (如：社團、比賽) 可一鍵切換為手動輸入。
  - **智慧判斷**：編輯時會自動偵測該科目是否存在於課表中，自動切換對應模式。
- **分數紀錄**：支援記錄該次作業或小考的得分 (滿分可自訂)，方便追蹤平時成績狀況。

### 💰 5. 學期記帳 (Accounting)

- **完整交易類型**：支援 **收入**、**支出** 與 **轉帳** (帳戶間互轉)。
- **資產帳戶管理**：
  - 可自訂支付方式 (如：現金、LINE Pay、悠遊卡、私房錢)。
  - 即時顯示各帳戶餘額。
- **圖表分析**：
  - **儀表板**：即時顯示本學期總收支與淨結餘。
  - **月報表**：長條圖分析每月收支變化。
  - **日統計**：詳細列出每日花費總額。

### 6. 🛠️ 豐富生活工具

- **行事曆**：視覺化月曆，自動計算並顯示「學期週次」。
- **幸運籤筒**：自訂選項隨機抽取，解決「午餐吃什麼」的選擇障礙。
- **紀念日倒數**：紀錄重要日子並自動計算天數。
- **學習進度條**：自訂讀書計畫目標 (頁數/章節)，視覺化追蹤完成率。

### ⚙️ 7. 系統與雲端 (System)

- **Firebase 雲端同步**：
  - 支援 Email、Google 登入與匿名試用。
  - 資料自動加密備份至雲端，跨裝置無縫接軌。
- **PWA 支援**：支援加入主畫面，提供類原生 App 的全螢幕操作體驗。
- **帳號安全**：支援密碼重設與帳號永久註銷 (Delete Account) 功能。

## 🛠️ 技術架構 (Tech Stack)

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla ES6+)
- **Backend**: Google Firebase (Firestore, Auth)
- **Tools**:
  - `Chart.js` (成績與記帳圖表)
  - `html2canvas` (課表匯出圖片)
